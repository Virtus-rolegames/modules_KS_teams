Вызывать функцию get_rooms. В нее передавать список юзверей. На выходе получать список списков юзверей распределенных по комнатам
Например:
  На входе:
    ['A', 'B', 'C', 'D', 'E', 'F', 'G', 'H', 'I', 'J', 'K', 'L', 'M', 'N', 'O', 'P', 'Q', 'R']
  На выходе:
    [['A', 'B', 'C', 'D', 'E'], ['F', 'G', 'H', 'I', 'J'], ['M', 'N', 'K', 'L'], ['Q', 'R', 'O', 'P']]
